# СВОИ — магазин оригинальной техники

Single-page storefront concept for **СВОИ** (svoistore). Static HTML/CSS/JS, no build step.

- `index.html` — the site (green design, primary frontend)
- `images/` — product & hero photos
- `design-system/` — design notes

## Local preview
Open `index.html` in a browser, or:
```bash
python3 -m http.server 8000
```

## Live demo
Hosted via GitHub Pages (root of `main`).
